## ⚠️ 重要声明（务必先读）

**本页面为汇报演示原型，页面内全部管网走向、缺陷点位、CCTV/QV 检测结果、管径材质建设年代、风险分值、人口户数等数据均为 `gen_data.py` 生成的示意 Mock 数据，不是巫山县城迁建区的真实普查/检测/勘察成果。**

- 迁建区、片区边界为**示意多边形**，不是正式测绘/GIS边界；
- 底图仅为空间参考，来自开源在线地图服务（CARTO Dark），不代表精确地理配准；
- 正式向甲方交付前，**必须将 `data/` 目录下的 GeoJSON/JSON 替换为实测普查、管网 CCTV/QV 检测、地质勘察、房屋鉴定及甲方确认的真实数据**。

页面右下角地图上及各信息模块均已保留相应文字提示，请勿删除。

## 一、这是什么

一个可本地运行的单页 Web 应用，用来向甲方（巫山县）汇报"管网缺陷如何联动影响坡体—挡墙—房屋—消防—居民安全"这一复合风险逻辑，而不是一张普通的排水管网 GIS 图。

核心能力：

1. 迁建区 14 个片区（朝云/翠屏/登龙/飞凤/集仙/净坛/起云/上升/神女/圣泉/松峦/宁江/聚鹤/西坪村）可点击查看"片区管网复合风险画像"；
2. 点击管线 → 查看该管段风险链（缺陷→渗漏/排水不畅→坡体滞水/挡墙变形→房屋开裂/内涝→居民安全风险）；
3. 点击缺陷点 → 查看缺陷详情、影响缓冲区、与边坡/房屋/内涝/消防的关联连线；
4. 内置"领导汇报演示模式"，一键按 6 步自动播放叙事逻辑；
5. 左侧图层开关可自由叠加/隐藏雨水管网、污水管网、混错接管段、箱涵、缺陷点、边坡/挡墙、房屋风险点、消防瓶颈点、内涝点。

## 三、如何本地运行

**必须通过本地 HTTP 服务打开，不能直接双击 index.html。**
（浏览器出于跨域安全策略，会拒绝 `file://` 协议下用 `fetch()` 读取 `data/*.json`。）

```bash
cd wushan-dashboard
python3 -m http.server 8000
# 然后浏览器打开 http://127.0.0.1:8000
```

或使用 Node：

```bash
npx serve .
```

推荐使用宽屏浏览器窗口（16:9，建议 ≥1600×900）以获得最佳汇报效果，可用浏览器全屏（F11）投影到会议大屏。

## 四、如何替换为真实数据

不需要改动 `index.html` / `app.js` 的任何代码，只需按下列字段结构替换 `data/` 目录下对应文件即可（详见十、数据结构约定）：

- `regions.geojson`：每个 Feature 的 `geometry` 换成甲方提供的正式片区边界，`properties` 中的人口、户数、管网长度、缺陷数等替换为真实统计值；
- `pipes.geojson`：`geometry` 换成管网普查/GIS 成果的真实线位，`diameter`/`material`/`buildYear` 等替换为管网普查真实字段；
- `defects.geojson`：替换为 CCTV/QV 检测报告中的真实缺陷点，`severity` 建议按检测报告的缺陷等级判定标准重新赋值；
- `relations.json`：建议在专业模型判定坡体/房屋/内涝/消防影响关系后，由技术负责人复核确认，而非沿用示意数据的随机生成逻辑；
- `overview.json`：可保留现有计算逻辑（脚本 `gen_data.py` 底部会自动汇总），或替换为人工审定的最终统计口径。

也可以直接修改 `gen_data.py` 顶部的 `BASE` 字典和生成规则，重新跑一遍脚本产出新的示意数据用于内部过程稿汇报。

## 五、底图与离线兜底

- 在线状态：使用 CARTO Dark（`dark_all`）瓦片，无需 API Key；
- 若检测不到网络瓦片响应（6 秒超时或 `tileerror`），页面自动切换为"底图离线模式"提示，并在地图区域叠加深色网格 + 光晕的示意底图，保证无网络环境下仍可完整演示交互逻辑；
- 若需要接入天地图 / 高德 / 百度等地图服务，请在 `app.js` 顶部 `CFG.tileUrl` 中替换为对应服务的瓦片地址模板，并将 Key 作为**独立配置项**（例如环境变量或单独的 `config.local.js`，不要提交到版本库），本项目默认未硬编码任何商业地图 API Key。

## 六、图表与依赖

- 地图引擎：[Leaflet 1.9.4](https://leafletjs.com/)（CDN 引入，无需注册/密钥）
- 图表引擎：[Apache ECharts 5.5](https://echarts.apache.org/)（CDN 引入）
- 均通过 `cdnjs.cloudflare.com` 加载；若汇报现场无外网，请提前下载对应版本的 `leaflet.js/css`、`echarts.min.js` 放入本地 `vendor/` 目录并修改 `index.html` 中的 `<script>`/`<link>` 引用路径。
